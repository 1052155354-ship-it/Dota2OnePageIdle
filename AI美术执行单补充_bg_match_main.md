## 二、主页面背景执行单

### 【asset_id】
bg_match_main

### 【资源名】
主页面背景

### 【资源用途】
用于单页面挂机放置游戏的整张主界面背景。该背景不是传统 MOBA 地图截图，也不是复杂战斗场景，而是服务于“炼金术士比赛成长”这一主题包装的单页面底图。它需要同时容纳顶部比赛信息、左侧状态区、右侧装备区、下方技能区和日志区，中间则留给炼金术士主立绘长期展示。

### 【输出目标】
一张 1920x1080 的 2D 游戏背景图，适合直接作为 Godot 主页面底图使用。画面完整、结构清楚、主题明确，能够支撑 UI 叠加，而不是抢占界面信息层。

### 【完整提示词】
一张 2D 游戏主界面背景图，用于 DOTA2 主题的一页挂机放置小游戏，主题围绕炼金术士的比赛成长。画面要传达“正在打一场炼金术士主题比赛”的感觉，但不要做成真实 MOBA 地图，不要做成复杂战斗场景，也不要做成电影概念图。整体是一个服务游戏 UI 的单页面背景，需要有明确的页面结构意识：中间区域相对开阔，方便后续摆放角色主体；顶部区域相对干净，方便叠加比赛时间和敌方基地血条；左侧和右侧留出相对平稳的视觉区域，方便叠加状态面板和装备区；下方留出较深色、较稳定的区域，方便叠加技能区和滚动日志区。

背景整体气质偏 DOTA 风格包装，但要简化为适合独立游戏原型的 2D 游戏背景。主题元素围绕炼金、毒雾、金币、工业实验感、比赛推进感展开。可以有低饱和的战场环境感、远处建筑轮廓、炼金器械、炼金烟雾、暗金与黄绿色能量痕迹，但都必须服务于“背景底图”这个功能，不要让细节太多，不要让中央区域变花，不要让边角出现大量抢眼物件。

整体色彩以暗绿色、黄绿色、暗金色、煤灰色、少量毒雾亮绿色为主，营造炼金术士主题氛围。画面需要有层次，但不要杂乱。建议中央略亮、四周略暗，方便后续把角色主体和 UI 叠上去。画面可以有轻微的比赛感、推进感、紧张感，但不能出现明确的战斗角色、不能出现多名英雄、不能出现复杂兵线交战主体。不要加入文字、UI、图标、血条、按钮，不要预先画任何具体 HUD 元素。

这是一张游戏背景资源，不是宣传海报。不要夸张透视，不要极端镜头，不要做成横版插画叙事场景。画面要平稳、可读、可承载 UI。最终目标是让这张图直接放入 Godot 后，能够稳定作为整张主页面背景，让玩家一眼感受到炼金术士主题比赛氛围，同时又不影响读数值和点击操作。

### 【负面约束】
不要角色主体，不要英雄立绘，不要多个角色，不要复杂战斗，不要地图截图，不要真实 DOTA 地图，不要纯海报，不要电影概念图，不要 3D 渲染，不要写实照片，不要文字，不要 UI，不要按钮，不要血条，不要图标，不要水印，不要边框，不要过强光效，不要让中央区域很花，不要过多高对比细节，不要让左右和下方没有 UI 留白。

### 【输出要求】
- 1920x1080
- PNG
- 完整背景图
- 不透明背景
- 适合 Godot 作为主页面底图
- 中央区域相对开阔
- 顶部、左右、下方预留 UI 区域

### 【验收重点】
1. 第一眼是否有炼金术士主题和比赛氛围。
2. 是否像游戏背景，而不是海报或概念图。
3. 中央是否足够干净，能放角色主体。
4. 左右和下方是否留出了足够稳定的 UI 区域。
5. 是否出现了多余角色、文字、UI、复杂战斗等不该有的内容。
